# Node.js and npm

Node.js and npm are major requirements for using the Web UI.

## Installation

npm is included in the installation of Node.js.

### On Ubuntu

```bash
curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt-get install -y nodejs
```

[Source](https://nodejs.org/en/download/package-manager/#debian-and-ubuntu-based-linux-distributions-enterprise-linux-fedora-and-snap-packages)

### On Windows

Download the latest LTS Release of Node.js from [here](https://nodejs.org/en/download/) - Choose the `Windows Installer (.msi)` for your Operating System type (32-bit vs 64-bit). Once the file is downloaded, double click on it to start the installation. The process is very simple, you can leave all the default options and simply press `Next` throughout the process.
